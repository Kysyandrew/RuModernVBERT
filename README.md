В данной папке все используемые файлы, данные после обработки https://drive.google.com/drive/folders/1VAOJXdw492M3-iLxVQiRfXR6aKfMcXSZ?usp=sharing

В репозитории находятся скрипты. Каждый из них можно запустить целиком, чтобы сделать следующее:
* mws_bench_vqa_to_vision_retrieval_eval.ipynb - адаптирует mwv_benvh под retrieval формат
* prepare_ocr_dense_baseline.ipynb - подготавливает ocr для финального замера ocr подхода
* retrieval_experiments.ipynb - формирует таблички финального замера
* train_lora_experiment_h3_l2.ipynb - пример запуска lora эксперимента (можно поменять только гиперпараметры)
* train_tokenizer_adaptation.ipynb - пример запуска lora эксперимента с учетом адаптации токенизатора (можно поменять только гиперпараметры) 
* translate_vidore_en_ru.ipynb - e2e перевод vidore датасета

Также в части ноутбуков сохранились выходы ячеек.
